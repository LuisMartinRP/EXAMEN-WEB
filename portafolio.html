Algoritmo kmeans.

Esta aplicacion desarrollada en lenguaje java implementa el algoritmo de identificacion kmeans el cual consta de dividir un banco de datos a partir de n numero de centroides y comenzar a moverlos de manera que cada dato al final quede referenciado a su centroide mas cercano. Los puntos son generados de manera aleatoria y la cantidad es decidida por el usuario al igual que el numero de centrides.

Algorimo genetico reninas en el tablero.

Esta aplicacion desarrollada en java resuelve el problema de n numero de reinas (n>=4) puestas en un tablero de ajedrez de n*n sin que se ataquen. Las coordenadas de las reinas se inicializan de manera aleatoria y conforme avanza el algoritmo cada vez la cantidad de ataques entre las reinas disminuye hasta que se dejan de atacar.
